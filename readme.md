# World Cup 2026 Dashboard

**[Live Site](https://keepitjibreel.github.io/worldcup-dashboard/)**

A lightweight live dashboard tracking all 104 matches of the 2026 FIFA World Cup across the USA, Canada, and Mexico.


## What it does

- Live scores, group standings, and upcoming fixtures for all 48 teams
- Clickable match cards with goalscorer data
- ML-powered win probability predictions for upcoming matches, updating automatically after every completed game
- Fully automated data pipeline — no manual updates required

## Tech stack

| Layer | Tools |
|---|---|
| Data pipeline | Python, pandas, requests |
| Predictions | scikit-learn (logistic regression) |
| Frontend | React, Vite |
| Deployment | GitHub Actions (automated), GitHub Pages (hosting) |
| Data source | [worldcup2026 API](https://github.com/rezarahiminia/worldcup2026) by Reza Rahiminia |

## Prediction model

Win probabilities are generated by a logistic regression model trained on 324 historical World Cup group stage matches (1994–2018). Features are the per-game difference between teams in points earned, goal differential, goals scored, and goals conceded — computed from live 2026 standings. Matchday-1 rows are down-weighted since teams have no prior form data. CV accuracy: ~63%.
